### Main HTML
```html

<script type="text/javascript">

```
## How to use
### TAG(arrays)
- -> arrays: TAG(...) or Text
- -> return: HTML-String
### TAG(string,arrays)
- -> string: HTML-tag name
- -> arrays: TAG(...) or Text
- -> return: HTML-String
### TAG(string,object,arrays)
- -> string: HTML-tag name
- -> object: Attributes
- -> arrays: TAG(...) or Text
- -> return: HTML-String
